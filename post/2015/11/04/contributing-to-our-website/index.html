Contributing to our website

2015-11-04

Our new sie has been updated and with it a improved workflow for collaboration. This site is powered by Hugo (a static site generator) that converts markdown files to html and allows for a quick and easy deployment to GitHub pages.

This allows for many potential contributors to simply perform a pull request and submit a markdown file alongside with the appropriate front matter (metadata about the content).

Should the pull request be accepted the code will automatically be pushed to the live website via Travis CI and published to intalgart.github.io.

For more information on how to contribute to the documentation site please consult the README.md file.
